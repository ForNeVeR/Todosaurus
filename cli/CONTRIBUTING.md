Contributor Guide
=================

Prerequisites
-------------
To work with the project, you'll need [.NET SDK 10][dotnet-sdk] or later.

Build
-----
Use the following shell command:

```console
$ cd cli
$ dotnet build
```

Test
----
Use the following shell command:

```console
$ cd cli
$ dotnet test
```

[dotnet-sdk]: https://dotnet.microsoft.com/en-us/download
